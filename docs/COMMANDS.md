## 캐릭터 관리

### !시트입력
캐릭터의 능력치를 설정하거나 수정합니다.

**사용법:**
```
!시트입력 "캐릭터 이름" [항목1] [값1] [항목2] [값2] ...
```

**예시:**
```
!시트입력 "첸 슈아이" 육체 3 감각 6 정신 4 사회 2
!시트입력 "첸 슈아이" 백병 5 회피 3 사격 4
!시트입력 "첸 슈아이" HP 24 침식률 35
```

**항목 종류:**
- **상위 능력치**: 육체, 감각, 정신, 사회
- **하위 스킬**: 백병, 회피, 사격, 지각, RC, 의지, 교섭, 조달
- **동적 스킬**: 운전:전차, 정보:컴퓨터, 예술:음악, 지식:역사 등
- **상태**: HP, 침식률, 침식D

### !빠른생성
JSON 형식으로 캐릭터를 한 번에 생성합니다. DX3 변환기와 연동하여 사용하세요.

**사용법:**
```
!빠른생성 [JSON데이터]
```

**예시 (축약형):**
```json
!빠른생성 {
  "n": "테스트캐릭터",
  "p": [["육체", 3], ["백병", 5]],
  "s": [["HP", 24], ["침식률", 30]],
  "cn": "蝴蝶",
  "e": "🦋",
  "sy": "엔젤 헤일로 × 모르페우스"
}
```

**JSON 필드:**
- `n` (name): 캐릭터 이름
- `p` (params): 능력치 배열
- `s` (status): 상태 배열
- `cn` (codeName): 코드네임
- `e` (emoji): 이모지
- `cv` (cover): 커버
- `w` (works): 웍스
- `br` (breed): 브리드 (퓨어/크로스/트라이)
- `sy` (syndromes): 신드롬
- `aw` (awakening): 각성
- `im` (impulse): 충동
- `c` (combos): 콤보 배열
- `l` (lois): 로이스 배열

### !지정
특정 캐릭터를 활성화합니다. 활성화된 캐릭터로 모든 명령어가 실행됩니다.

**사용법:**
```
!지정 "캐릭터 이름"
```

**예시:**
```
!지정 "첸 슈아이"
!지정 첸슈아이
```

### !지정해제
현재 활성화된 캐릭터를 해제합니다.

**사용법:**
```
!지정해제
```

### !시트확인
현재 활성 캐릭터의 모든 정보를 확인합니다.

**사용법:**
```
!시트확인
```

**출력 예시:**
```
🦋 첸 슈아이 :: 「蝴蝶」
> 고등학생｜UGN
> CROSS｜ANGEL HALO × MORPHEUS
> 꿈의 사도｜호기심
> D-Lois｜No.98 Legacy: Dream of Abyssal City

> HP 24  |  침식률 35  |  침식D 0  |  로이스 3

>     【육체】  3   백병: 5 회피: 3
>     【감각】  6   사격: 4 지각: 5
>     【정신】  4   RC: 3 의지: 4
>     【사회】  2   교섭: 1 조달: 2
```

## 캐릭터 속성 설정

### !코드네임
캐릭터의 코드네임을 설정합니다.

**사용법:**
```
!코드네임 "코드네임"
```

**예시:**
```
!코드네임 "蝴蝶"
!코드네임 Butterfly
```

### !이모지
캐릭터의 대표 이모지를 설정합니다.

**사용법:**
```
!이모지 [이모지]
```

**예시:**
```
!이모지 🦋
!이모지 ⚔️
```

### !커버
캐릭터의 커버(직업/신분)를 설정합니다.

**사용법:**
```
!커버 [커버명]
```

**예시:**
```
!커버 고등학생
!커버 회사원
```

### !웍스
캐릭터가 소속된 웍스를 설정합니다.

**사용법:**
```
!웍스 [웍스명]
```

**예시:**
```
!웍스 UGN
!웍스 FH
```

### !브리드
캐릭터의 브리드를 설정합니다.

**사용법:**
```
!브리드 [퓨어|크로스|트라이]
```

**예시:**
```
!브리드 크로스
!브리드 트라이
```

### !신드롬
캐릭터의 신드롬을 설정합니다 (최대 3개).

**사용법:**
```
!신드롬 [신드롬1] [신드롬2] [신드롬3]
```

**예시:**
```
!신드롬 엔젤헤일로
!신드롬 엔젤헤일로 모르페우스
!신드롬 발로르 블랙독 브람스토커
```

**지원 신드롬:**
- 엔젤헤일로, 발로르, 블랙독, 브람스토커, 키마이라, 엑자일
- 하누만, 모르페우스, 노이만, 오르쿠스, 샐러맨더, 솔라리스
- 우로보로스, 아자토스, 미스틸테인, 글레이프닐

### !각성
캐릭터의 각성을 설정합니다.

**사용법:**
```
!각성 [각성명]
```

**예시:**
```
!각성 꿈의 사도
!각성 피의 전사
```

### !충동
캐릭터의 충동을 설정합니다.

**사용법:**
```
!충동 [충동명]
```

**예시:**
```
!충동 호기심
```

### !D로

**사용법:**
```
!D로 [번호] [이름]
```

**예시:**

## 판정 시스템

### !판정
능력치 기반 판정을 실행합니다. 침식D가 자동으로 적용됩니다.

**사용법:**
```
!판정 [항목]
```

**예시:**
```
!판정 백병          → (3+0)dx+5 백병 판정
!판정 육체          → (3+0)dx+0 육체 판정
!판정 정보:컴퓨터   → (2+0)dx+3 정보:컴퓨터 판정
```

**판정 가능 항목:**
- 상위 능력치: 육체, 감각, 정신, 사회
- 하위 스킬: 백병, 회피, 사격, 지각, RC, 의지, 교섭, 조달
- 동적 스킬: 운전:전차, 정보:컴퓨터, 예술:음악, 지식:역사 등

## 전투 시스템

### !등침 / !등장침식
등장 시 1d10을 굴려 침식률을 추가합니다.

**사용법:**
```
!등침
!등장침식
```

**예시:**
```
사용자: !등침
봇: 1d10 등장침식 @사용자
[주사위봇이 7을 굴림]
봇: 첸 슈아이 등장침식 +7 → 현재 침식률: 42
```

### 상태 변경
HP, 침식률 등의 상태를 변경합니다.

**사용법:**
```
![상태]+[값]    # 증가
![상태]-[값]    # 감소
![상태]=[값]    # 설정
```

**예시:**
```
!HP-10          → HP가 10 감소
!침식률+5       → 침식률이 5 증가
!육체=4         → 육체를 4로 설정
```

**침식D 자동 조정:**
- 침식률 60 이상: 침식D +1
- 침식률 80 이상: 침식D +2
- 침식률 100 이상: 침식D +3
- 침식률 130 이상: 침식D +4
- 침식률 190 이상: 침식D +5

## 콤보 시스템

### !콤보
침식률 조건부 콤보를 저장합니다.

**사용법:**
```
!콤보 "콤보 이름" [침식률 조건] [콤보 데이터]
```

**침식률 조건:**
- `99↓`: 침식률 99 이하일 때
- `100↑`: 침식률 100 이상일 때
- `130↑`: 침식률 130 이상일 때

**예시:**
```
!콤보 "연속 사격" 99↓ 《C: 발로르(2) + 흑의 철퇴(4)》
!콤보 "연속 사격" 100↑ 《C: 발로르(2) + 흑의 철퇴(4) + 광전사(5)》
!콤보 "필살기" 130↑ 《M: 샐러맨더(7) + 불의 심판(10) + 완전마인화》
```

### !@"콤보이름"
현재 침식률에 맞는 콤보를 자동으로 검색하여 출력합니다.

**사용법:**
```
!@"콤보 이름"
```

**예시:**
```
사용자: !@"연속 사격"
봇: > 99↓ 【연속 사격】
    > 《C: 발로르(2) + 흑의 철퇴(4)》
```

### !콤보삭제
저장된 콤보를 삭제합니다.

**사용법:**
```
!콤보삭제 "콤보 이름"
```

**예시:**
```
!콤보삭제 "연속 사격"
```

## 로이스 시스템

### !로이스
새로운 로이스를 추가합니다. P감정에 `*`를 붙이면 메인 감정으로 표시됩니다.

**사용법:**
```
!로이스 "이름" [P감정] [N감정] [내용]
```

**예시:**
```
!로이스 "배신자" 증오* 분노 나를 배신한 동료
!로이스 "소중한 사람" 애정* 집착 지키고 싶은 사람
!로이스 "라이벌" 경쟁심 증오 언젠가 넘어서고 싶은 존재
```

**출력 예시:**
```
✅ 첨 슈아이의 로이스 "배신자"가 등록되었습니다.
【P: 증오】 / N: 분노
나를 배신한 동료
```

### !로이스삭제
로이스를 삭제합니다.

**사용법:**
```
!로이스삭제 "이름"
```

**예시:**
```
!로이스삭제 "배신자"
```

### !타이터스
로이스를 타이터스로 변환합니다. 침식률이 +5 상승하고 로이스가 삭제됩니다.

**사용법:**
```
!타이터스 "이름"
```

**예시:**
```
사용자: !타이터스 "배신자"
봇: 🔥 첸 슈아이의 로이스 "배신자"가 타이터스로 변환되었습니다!
```

## 관리 명령어

### !리셋
캐릭터 데이터를 초기화합니다.

**사용법:**
```
!리셋              # 전체 데이터 초기화
!리셋 콤보         # 콤보만 초기화
!리셋 로이스       # 로이스만 초기화
!리셋 이펙트       # 이펙트만 초기화
!리셋 [항목명]     # 특정 항목만 초기화
```

**예시:**
```
!리셋              # 모든 데이터 삭제
!리셋 콤보         # 모든 콤보 삭제
!리셋 HP           # HP 값만 삭제
```

### !캐릭터삭제
캐릭터를 완전히 삭제합니다.

**사용법:**
```
!캐릭터삭제 "캐릭터 이름"
```

**예시:**
```
!캐릭터삭제 "첸 슈아이"
```

### !도움
봇의 도움말을 표시합니다.

**사용법:**
```
!도움
```

### !업데이트 (봇 관리자 전용)
봇 버전을 업데이트하고 모든 서버에 공지합니다.

**사용법:**
```
!업데이트 [major|minor|patch] [공지 메시지]
```

**예시:**
```
!업데이트 minor 새로운 콤보 시스템이 추가되었습니다!
!업데이트 patch 버그 수정 및 안정성 개선
```

## 팁 💡

### 1. 따옴표 사용
긴 이름이나 공백이 포함된 경우 따옴표를 사용하세요:
```
!지정 "첸 슈아이"
!콤보 "연속 공격" 99↓ 콤보내용
```

### 2. 침식D 자동 계산
침식률이 변경되면 침식D가 자동으로 조정됩니다:
```
사용자: !침식률+30
봇: ⚠️ 침식률이 100이 되어 침식D가 2 → 3으로 상승했습니다!
```

### 3. 빠른생성 활용
DX3 변환기를 사용하면 캐릭터를 빠르게 생성할 수 있습니다!

### 4. 콤보 조건 활용
같은 이름의 콤보에 여러 조건을 설정하여 상황별 콤보를 관리하세요:
```
!콤보 "기본공격" 99↓ 약한 버전
!콤보 "기본공격" 100↑ 강한 버전
!콤보 "기본공격" 130↑ 최강 버전
```

## 문제 해결 🔧

### 명령어가 작동하지 않을 때
1. `!지정 "캐릭터명"`으로 캐릭터를 먼저 지정했는지 확인
2. 명령어 철자가 정확한지 확인
3. 필요한 매개변수를 모두 입력했는지 확인

### 침식D가 올라가지 않을 때
침식률을 직접 변경하면 자동으로 침식D가 계산됩니다:
```
!침식률=100
```

### 데이터가 저장되지 않을 때
봇이 정상적으로 실행 중인지 확인하고, 봇 관리자에게 문의하세요.
